## Customization

You can easily customize the website by modifying:

- **Colors**: Edit CSS variables in `styles.css` (`:root` section)
- **Content**: Update text in `index.html`
- **Styles**: Modify `styles.css` for design changes
- **Interactivity**: Adjust `script.js` for additional features
